# Handoff — 2026-03-19

## 当前结论
`openclaw-webchat` 当前应继续以 `main` 为唯一开发基线。2026-03-19 这一轮已把公开化整理、访问与安全设置、轻认证、设置内重启、设置面板重组，以及简体中文 / English 界面切换并入主线。

当前已确认的主线状态：
- 当前版本已提升到 `0.1.4`
- 默认服务地址仍为 `http://127.0.0.1:3770`
- 当前开发基点仍为 `main`
- 用户级 LaunchAgent `ai.openclaw.webchat` 继续作为常驻运行方式
- 当前主线新增的公开化与运行方式整理：
  - README、CHANGELOG、CONTRIBUTING、SECURITY、SECURITY_MODEL、最小 GitHub CI 已补齐
  - 设置面板已拆成一级手风琴分区：联系人、外观、访问与安全、关于、手动启动服务
  - 关于页已提供项目简介与 GitHub 仓库链接
  - 手动启动服务分区已提供安装、启动、重启命令提示
- 当前主线新增的访问与安全能力：
  - 支持本机 / 局域网 / Tailscale 对应的访问模式设置
  - 支持可选轻认证
  - 访问方式切换支持设置页内触发服务重启
  - 文档访问范围说明已改为跟随 OpenClaw 当前配置
- 当前主线新增的隐私与媒体链路修正：
  - 上传接口不再把绝对路径返回到浏览器
  - 上传源改为 opaque handle
  - Baichai 等 agent 产出的本地图片恢复显示，不再被 WebChat 额外路径范围限制挡住
- 当前主线新增的界面国际化：
  - 外观分区已开放语种选择
  - 当前支持简体中文 / English
  - 语种偏好保存在浏览器本地
