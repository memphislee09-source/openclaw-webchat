手动启动服务

如果你需要手动启动或恢复服务，可以按下面的步骤操作。

1. 项目目录
先进入 openclaw-webchat 项目目录
2. 首次安装依赖
npm install
3. 启动服务
npm start
4. 已注册 LaunchAgent 时可直接重启
启动后这里会显示可用重启命令
